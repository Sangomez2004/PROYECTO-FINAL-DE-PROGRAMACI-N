# Proyecto Parqués en Python

## Descripción
Este proyecto implementa un juego de Parqués utilizando Python y la biblioteca Pygame.

## Requisitos
1. Tener instalado **Python 3.x** en el sistema.
2. Instalar la biblioteca necesaria (**Pygame**).

## Instalación de Dependencias
Antes de ejecutar el programa, instala Pygame con el siguiente comando:

pip install pygame

## Instrucciones de Ejecución

### Opción 1: Ejecutar desde la Terminal (Windows/Linux/Mac)
1. **Descargar el código**  
   - Si usas Git, clona el repositorio con:  
     git clone <URL_DEL_REPOSITORIO>
     cd <nombre_del_proyecto>
   - O descarga el archivo `Cógigo_final_paques_UN.py` manualmente.

2. **Abrir la Terminal y ubicar la carpeta del proyecto**  
   - En Windows:  
     cd Ruta\de\la\carpeta
   - En Mac/Linux:  
     cd /ruta/de/la/carpeta

3. **Ejecutar el programa**  
   python Cógigo_final_paques_UN.py

### Opción 2: Ejecutar desde un Entorno de Desarrollo (PyCharm, VS Code, IDLE)
1. Abre el archivo `Cógigo_final_paques_UN.py` en el editor.
2. Instala Pygame si no está instalado.
3. Ejecuta el código desde el botón **Run**.

## Notas Adicionales
- Para salir del juego, cierra la ventana de Pygame o presiona `Ctrl + C` en la terminal.
- Asegúrate de ejecutar el programa en una resolución adecuada para visualizar bien el tablero.
